# 실습 3 — 강의 표지 이미지 만들기 (15분)

[목차로 돌아가기](../README.md)

내가 하고 싶은 강의의 홍보 표지 이미지를 만듭니다. 잘 나온 이미지는 저장해 두세요. 7/10(금) 콘텐츠 제작 차시에 이어서 씁니다.

도구: 구글 제미나이(이미지 생성) · Canva · 미리캔버스 AI 등. 실습은 제미나이로 진행합니다.

---

## 이미지 프롬프트 공식

```
주제  +  스타일  +  구도·배경  +  분위기·용도
```

| 칸 | 무엇을 | 예시 |
|---|---|---|
| 주제 | 무엇을 그릴까 | 강의하는 중년 여성 |
| 스타일 | 그림체·화풍 | 밝은 수채화풍 |
| 구도·배경 | 어디서, 어떻게 | 밝은 강의실, 정면 |
| 분위기·용도 | 느낌과 쓰임새 | 따뜻한 느낌, 강의 표지용 |

---

## 복사해서 써 볼 예시

```
밝은 강의실에서 수강생을 향해 환하게 강의하는 친근한 중년 여성,
밝은 수채화풍, 따뜻하고 활기찬 분위기,
강의 홍보 표지용, 상단에 제목을 넣을 여백을 남겨 주세요.
```

아쉬우면 말로 고칩니다.

```
좋아요. 배경을 조금 더 밝게 하고, 인물을 더 자신감 있는 표정으로 바꿔 주세요.
```

---

## 실습 진행

1. 내 강의 주제를 한 줄로 정합니다
2. 공식(주제 + 스타일 + 구도 + 분위기 + 용도)으로 묘사문을 씁니다
3. 생성한 뒤 아쉬운 점을 말로 고쳐 두 번 다시 만듭니다
4. 마음에 드는 이미지를 저장합니다

---

## 잘 나오게 하는 팁

- 글자는 이미지에 넣지 않습니다. AI 이미지의 글자는 자주 깨집니다. 제목 넣을 여백만 요청하고, 글자는 나중에 캔바에서 얹습니다.
- 실존 연예인이나 특정 작가 화풍은 요청하지 않습니다. 초상권·저작권 문제가 있습니다. ([저작권과 AI 윤리](../자료/저작권과_AI윤리.md))
- 이미지 생성이 안 되거나 한도가 걸리면 강사 시연을 함께 보거나 Canva·미리캔버스 AI로 대신 만듭니다.
