## 자기소개 · 강사 소개글
```
당신은 강사 소개글 전문 카피라이터입니다.
나는 [경력]을 해 왔고 앞으로 [분야] 강사로 활동하려 합니다.
수강생이 배우고 싶어질 소개글을 300자 내외로,
따뜻하고 신뢰감 있는 어조로 써 주세요.
```

## 자기소개서 · 이력서 (성과 강조)
```
당신은 중장년 재취업 컨설턴트입니다.
나는 [경력]을 해 왔고 [직무]에 지원합니다.
강사 직무 기준으로 내 경력 강점 3개를 성과가 드러나게 뽑아 주세요.
그다음 그것을 녹여 자기소개서 초안을 600자로 써 주세요.
```

## 강의 기획 (단계별)
```
1) 시니어 대상 [주제] 강의 주제 5개를 제안해 줘.
2) 그중 3번으로 2시간 강의 목차를 짜 줘.
3) 목차 중 1교시의 상세 진행안을 표로 정리해 줘.
```

## 블로그 · SNS
```
50대 이웃에게 말하듯 친근한 존댓말로,
[주제]에 대한 블로그 글을 써 줘. 400자 내외로.
```

## 수강생 모집 문구
```
[강의 이름] 수강생 모집 글을 써 줘.
핵심 혜택 3가지가 드러나게, 친근한 존댓말로 200자 내외로.
모집 문구는 3개 버전으로 제안해 줘.
```

## 보고서 · 제안서 (개조식)
```
[주제]에 대한 기획안을 개조식으로 정리해 줘.
배경 – 목적 – 내용 – 기대효과 순서로.
```

## 이미지 (표지)
```
[상황]에서 [대상]이 [행동]하는 모습, [화풍], [분위기],
강의 홍보 표지용, 상단에 제목 넣을 여백 포함.
```

- 확실하지 않은 부분은 표시해 주세요
- 한국어로 답해 주세요
