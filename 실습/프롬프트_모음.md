## 자기소개 · 강사 소개글

```
당신은 강사 소개글 전문 카피라이터입니다.
나는 [경력]을 해 왔고 앞으로 [분야] 강사로 활동하려 합니다.
수강생이 배우고 싶어질 소개글을 300자 내외로,
따뜻하고 신뢰감 있는 어조로 써 주세요.
```

## 자기소개서 · 이력서 (성과 강조)

```
당신은 중장년 재취업 컨설턴트입니다.
나는 [경력]을 해 왔고 [직무]에 지원합니다.
강사 직무 기준으로 내 경력 강점 3개를 성과가 드러나게 뽑아 주세요.
그다음 그것을 녹여 자기소개서 초안을 600자로 써 주세요.
```

## 강의 기획 (단계별)

```
1) 시니어 대상 [주제] 강의 주제 5개를 제안해 줘.
2) 그중 3번으로 2시간 강의 목차를 짜 줘.
3) 목차 중 1교시의 상세 진행안을 표로 정리해 줘.
```

```

```


## 재취업·실무에 바로 쓰는 (참고: 비즈니스 프롬프트 교재)

예상 면접 질문 뽑기
```
당신은 채용 면접관입니다. 나는 [경력]으로 [직무]에 지원합니다.
실제로 나올 만한 면접 질문 10개와, 각 질문의 의도를 함께 알려 주세요.
```

이력서 강점 도출
```
아래 내 경력에서 [직무]에 강점이 되는 3가지를 성과가 드러나게 뽑아 주세요.
그다음 이력서에 넣을 한 줄 문장으로 각각 정리해 주세요.
경력: [붙여넣기]
```

문장·맞춤법 다듬기
```
아래 글의 맞춤법과 어색한 표현만 고쳐 주세요. 뜻은 바꾸지 말고,
무엇을 왜 고쳤는지 표로 정리해 주세요.
글: [붙여넣기]
```

일일 보고서·회의 정리
```
아래 메모를 배경-진행-결과-다음 할 일 순서로 정리해 주세요. A4 반 장 분량으로.
메모: [붙여넣기]
```

고객·수강생 응대 FAQ
```
[강의/서비스 이름]에 대해 수강생이 자주 물을 질문 10개와 답변을 만들어 주세요.
답은 친근한 존댓말로, 3~4줄 이내로.
```

막힐 때 — AI에게 프롬프트를 부탁하기 (메타 프롬프트)
```
나는 [상황]입니다. [원하는 결과]를 잘 얻으려면 어떻게 부탁하면 좋을지,
그 부탁(프롬프트)을 대신 만들어 주세요.
```
