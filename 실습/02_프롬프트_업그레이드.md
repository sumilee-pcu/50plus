실습 1에서 쓴 프롬프트를 네 가지 기법으로 개선합니다.

---

## 목표

한 번에 안 되면 대화를 이어가며 고칩니다. 실습 1의 기본 프롬프트를 아래 네 가지 기법 중 두 가지 이상으로 개선해, 결과가 좋아지는 것을 확인합니다.

---

## 네 가지 기법

### 기법 1 — 예시 제공

원하는 느낌의 글을 한두 개 붙여 넣습니다. AI가 그 문체와 길이를 그대로 흉내 냅니다. 말로 설명하기보다 예시 하나를 붙이는 편이 정확합니다. 이렇게 예시로 방향을 잡아 주는 것을 퓨샷(few-shot)이라고 하며, 별도 학습 없이 예시만으로 결과가 달라지는 성질입니다.

```
아래는 제가 좋아하는 강사 소개글입니다.
[30년 현장 경험을 교실로 가져오는 강사, ○○○입니다. …]
이 글과 같은 어조와 길이로 제 소개글을 써 주세요. 제 경력은…
```

### 기법 2 — 단계별 지시

통째로 시키기보다 나눠서 시키면 결과가 좋아집니다.

```
STEP 1  시니어 대상 스마트폰 교육, 강의 주제 5개를 제안해 줘
STEP 2  3번 주제로 2시간 강의 목차를 짜 줘
STEP 3  목차 중 1교시의 상세 진행안을 써 줘
```

앞의 답을 이어받아 다음을 시키는 것, 이것이 대화형 AI의 가장 큰 힘입니다. 한 단계씩 생각하게 하면 정확도가 오릅니다. `한 단계씩 생각해 보자` 한 문장을 더한 것만으로 산술 문제 정답률이 17.7%에서 78.7%로 오른 연구도 있습니다(Kojima et al., 2022). 근거는 [이론 근거](../자료/이론_근거.md) 참고.

### 기법 3 — 반복 개선

```
초안 받기  →  피드백 주기  →  다시 받기 (만족할 때까지 2~3회)
          더 짧게 / 사례를 넣어 / 어조를 부드럽게
```

아쉽다는 느낌에서 멈추지 말고, 무엇이 어떻게 아쉬운지 말로 알려주면 AI는 그대로 고쳐 옵니다.

### 기법 4 — 형식 제어와 검증

```
표로 정리해 줘 (항목 / 내용 / 비고)
체크리스트 10개로 만들어 줘
초등학생도 이해할 쉬운 말로
A4 한 장 분량, 개조식으로
```

검증 요청

```
방금 답에서 확실하지 않은 부분이 있으면 스스로 표시해 줘
근거가 되는 출처를 함께 알려 줘
```

AI는 모르는 것도 그럴듯하게 지어냅니다. 이것을 할루시네이션이라고 합니다. 숫자·이름·법령은 꼭 다시 확인합니다.

## 실습 진행

BEFORE — 실습 1의 첫 프롬프트 (역할·맥락만 있는 기본형)

AFTER — 아래 문구 중 두 가지 이상 골라 추가

- 예시: 이 글과 같은 어조로 다시 써 줘
- 단계: 강점 3개 먼저 → 한 문단으로
- 피드백: 더 짧고 자연스럽게 고쳐 줘
- 형식: 표로 정리해 줘 (항목 / 내용)

복사해서 써 볼 예시 (예시 제공 + 단계별 지시 결합)

```
아래는 제가 좋아하는 강사 소개글입니다.
[30년 현장 경험을 교실로 가져오는 강사, ○○○입니다. …]
이 글과 같은 따뜻한 어조와 비슷한 길이로 제 소개글을 써 주세요.
먼저 제 강점 3가지를 뽑아 정리한 다음, 그것을 녹여 한 문단으로 완성해 주세요.
저는 25년간 백화점 고객 상담을 해왔고, 시니어 서비스 교육 강사에 지원합니다.
```
