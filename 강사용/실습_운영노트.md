# 강사용 실습 운영 노트

[목차로 돌아가기](../README.md)

강사(이수미)가 당일 3개 실습과 토론을 진행하기 위한 운영 대본입니다. 배포용 인쇄물은 [수강생 워크시트](../README.md)를, 상세 대본은 원본 docx를 참고하세요.

### 실습 운영 세 원칙 (중장년 수강생)

1. 먼저 보여주고, 따라 하게 한다. 모든 실습은 강사가 화면으로 1회 시연한 뒤 시작한다.
2. 완성이 아니라 경험이 목표다. 한 번 보내보고 한 번 고쳐보는 데까지면 성공이다.
3. 손 들면 바로 간다. 보조강사가 통로를 돌며 로그인·오타·막힘을 즉시 해결한다.

## 수업 전 점검 (도착~13:00)

- [ ] 강사 화면이 빔프로젝터에 크게 보이는지 (브라우저 `Ctrl + +`로 확대)
- [ ] 강사 제미나이(Gemini) 구글 로그인 상태, 새 대화 준비
- [ ] 이미지 생성 되는 계정인지 미리 1회 테스트 (실습 3 대비, 무료 계정은 한도 있음)
- [ ] 수강생 실습 방식 결정: 각자 기기 또는 2인 1조 (보조강사와 합의)
- [ ] 워크시트 인쇄 배포, 필기구 준비
- [ ] 시연용 예시 프롬프트를 메모장에 미리 복사

보조강사와 3분 미팅: 손 들면 보조강사가 먼저 대응하고 3분 넘으면 강사에게 공유, 로그인 화면 미리 준비, 실습 종료 2분 전 신호.

## 실습 1 첫 프롬프트 (15분)

시연 대본(2분): 아쉬운 예 `재취업 자기소개서 써줘` 입력 → 밋밋한 결과 → 4요소 넣은 좋은 예 입력 → 비교 → 이제 여러분 차례라고 넘긴다.

시연용 검증 프롬프트

```
당신은 중장년 재취업 전문 컨설턴트입니다.
나는 25년간 백화점에서 고객 상담과 매장 관리를 해왔고,
이제 시니어 대상 서비스 교육 강사로 활동하려 합니다.
이 경력이 강사 직무에 왜 강점이 되는지가 드러나도록,
저를 소개하는 글 한 단락을 300자 내외로 정중한 존댓말로 써 주세요.
```

자주 나오는 문제와 대응

| 문제 | 대응 |
|---|---|
| 로그인 안 됨/계정 없음 | 구글 로그인 우회, 정 안 되면 2인 1조 |
| 뭘 쓸지 막막 | 가장 오래 한 일을 한 문장으로 → 맥락 칸에 |
| 영어로 나옴 | `한국어로 답해 주세요` 추가 |
| 너무 길다/딱딱 | `300자로 줄여`, `더 따뜻하게` → 다음 실습 예고 |

보조강사: 가장 손이 많이 가는 실습. 로그인·새 대화를 1:1로 해결. 빈칸 못 채우면 질문으로 재료를 꺼낸다.

## 실습 2 프롬프트 업그레이드 (20분)

시연 대본(3분), 반복 개선을 라이브로: 실습 1 결과에 `더 짧게, 사례 하나 넣어, 어조는 부드럽게 고쳐 주세요` → 변화 확인 → `표로 정리해 주세요(항목/내용)`.

시연용 검증 프롬프트 (예시 제공 + 단계별 지시)

```
아래는 제가 좋아하는 강사 소개글입니다.
[30년 현장 경험을 교실로 가져오는 강사, ○○○입니다. …]
이 글과 같은 따뜻한 어조와 비슷한 길이로 제 소개글을 써 주세요.
먼저 제 강점 3가지를 뽑아 정리한 다음, 그것을 녹여 한 문단으로 완성해 주세요.
저는 25년간 백화점 고객 상담을 해왔고, 시니어 서비스 교육 강사에 지원합니다.
```

자주 나오는 문제와 대응

| 문제 | 대응 |
|---|---|
| 기법 다 넣다 뒤엉킴 | 한 번에 하나씩. 예시 하나 먼저, 그다음 형식 |
| 고쳐도 안 바뀜 | 구체어로. 딱딱하다 대신 친구에게 말하듯 편한 존댓말로 |
| 지어낸 정보 보임 | 확실하지 않은 부분 표시해 달라고 요청, 숫자·이름 재확인 |

보조강사: 두 가지만 골라 넣기를 반복 안내. 잘 나온 사례는 강사에게 귀띔.

## 실습 3 강의 표지 이미지 (15분)

시연 대본(2분): 공식 5칸 채우며 한 문장으로 합치기 → 생성 → 수정 요청 1회 시연 → 저장 방법 안내.

시연용 검증 프롬프트

```
밝은 강의실에서 수강생을 향해 환하게 강의하는 친근한 중년 여성,
밝은 수채화풍, 따뜻하고 활기찬 분위기,
강의 홍보 표지용, 상단에 제목을 넣을 여백을 남겨 주세요.
```

자주 나오는 문제와 대응

| 문제 | 대응 |
|---|---|
| 생성 안 됨/한도 초과 | 강사 대표 시연 또는 2인 관찰조, Canva·미리캔버스 대안 |
| 글자가 깨짐 | 제목 여백만 요청, 글자는 나중에 캔바에서 |
| 실존 인물·작가 화풍 요구 | 저작권·초상권 위험, 일반 묘사로 전환 |

보조강사: 저장을 1:1로 지원(다음 차시 재료). 이미지 안 되는 좌석은 2인 관찰조로.

## 토론 운영 (PART 4)

정답이 아니라 강사로서의 기준을 세우는 자리로 진행한다.

- Q1. AI 교안으로 강의료. 나라면 수강생에게 어떻게 밝히겠는가로 좁힌다. 활용 사실 고지, 최종 책임은 강사.
- Q2. 틀린 정보를 배워 갔다면 책임은. 강사·도구·수강생 세 주체. 숫자·법령은 원 출처 재확인, 변호사 사례.

마무리: 오늘의 네 가지(4요소·4기법·경력 활용·검증과 저작권)를 되짚고, 실습 3 이미지를 7/10 준비물로 안내한다.
