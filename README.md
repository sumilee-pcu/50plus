## 시작하기 — 제미나이 열기

준비물: 본인 노트북, 구글 ID 계정

1. gemini.google.com 접속 또는 제미나이 앱 열기
2. 내 구글 계정으로 로그인. 이미 로그인돼 있으면 바로 사용, 안 되면 손을 들어 주세요
3. 대화창에 입력하면 준비 끝

막힐 때는 이렇게 합니다.

- 답이 영어로 오면 끝에 `한국어로 답해 주세요` 를 붙입니다
- 글이 길거나 딱딱하면 `더 짧게`, `더 따뜻하게 고쳐 주세요` 라고 합니다
- 수강생이나 타인의 개인정보, 실제 사연은 입력하지 않습니다
