## 시작하기 — ChatGPT 열기

**준비물: 본인 노트북 + 구글 ID 계정**

1. `chatgpt.com` 접속 또는 **ChatGPT 앱** 열기
2. **‘구글 계정으로 로그인’** 을 누르고 내 구글 계정으로 로그인 — 안 되면 손을 들어 주세요

**막히면 이렇게**
- 답이 영어로 오면 → 끝에 **“한국어로 답해 주세요”**
